Orleans.Serialization.NodaTime is a library that provides Orleans serialization functionality for NodaTime.

The library should implement codecs and copiers for the following types:

- [x] Instant
- [x] Offset
- [x] CalendarSystem
- [x] LocalDateTime
- [x] LocalDate
- [x] LocalTime
- [ ] OffsetDateTime
- [ ] OffsetDate
- [ ] OffsetTime
- [x] DateTimeZone
- [ ] ZonedDateTime
- [ ] Duration
- [ ] Period
- [ ] Interval
